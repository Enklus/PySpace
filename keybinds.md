### moving
| Key  | Function |
|------------|----------:|
| W,A,S,D    | move around |
| SPACE | move up |
| left CTRL   |     move down |
| left SHIFT | slow movement by x10 |
| numpad + | increase speed |
| numpad - | decrease speed |

### looking
| Key  | Function |
|------------|----------:|
| X | lock X axis on mouse movement |
| Y | lock Y axis on mouse movement |
| RETURN | rectify axis |

### recording
| Key  | Function |
|------------|----------:|
| R | start/stop recording |
| P | start playback and saving of video |
| C | screenshot |
